Lithic Artifact Data

The Lithic Artifact Database contains observations and measurements for all analyzed stone artifacts recovered at the Fredricks site. The database is in dBase III format (*.dbf) and can be accessed using most quality database and spreadsheet applications.

Download Lithic Artifact Database (745 KB)

Description of the field headings used in the Lithic Artifact Database.

Heading Description
PKEY A technical field used by the report software on the original CD-ROM.
CATALOG_NO The Research Laboratories of Archaeology's catalog number of the artifact.
CATEGORY Category (i.e., artifact type).
BLANK Blank.
WORKING_ED Working edge.
RAW_MATERI Raw material.
CONDITION Condition of artifact (this field is not displayed in the report software, but does exist in the data file for analysis purposes).
SIZE Size category.
COUNT Count.
LENGTH_MM Length (mm).
WIDTH_MM Width (mm).
THICKNESS_ Thickness (mm).
EDGES Configuration of tool edges.
EDGE1_SHAP Shape of first edge (this field is not displayed in the report software, but does exist in the data file for analysis purposes).
EDGE2_SHAP Shape of second edge (this field is not displayed in the report software, but does exist in the data file for analysis purposes).
BASAL_SHAP Basal shape.
PATINATION Degree of patination (this field is not displayed in the report software, but does exist in the data file for analysis purposes).
OADINFO A technical field used by the report software on the original CD-ROM.
ADINFO Context.

Version Information

The authors of Excavating Occaneechi Town (Web edition) recognize that, while all the information recovered in an archaeological excavation has scientific importance, it may be inappropriate to display or view some of that information, such as photographs of human bones, in a public or instructional context. Therefore, we have provided two versions of this edition. The Professional Version provides access to all of the information about the site; the Instructional Version provides access to all of the information except for photographs of human burials.

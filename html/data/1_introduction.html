Introduction

While the authors of the report have invested a great deal of time in the report software to make it as easy to use and as powerful as possible, there is no way that we can anticipate all of the ways that archaeologists and researchers might want to use the information contained in Excavating Occaneechi Town. The solution that we have chosen is to provide all of the raw data from this report in easy-to-use, widely accepted formats.

The use of any of this information will require separate third-party software. The types of software that you might need include a word processor, spreadsheet, database manager, image viewer, and computer-aided design (CAD) program. Accessing this information will also require some knowledge of how to use the software.

Here are just a few of the reasons why you might want to access these separate data files:

  • Print out the maps at a different resolution.
  • Print out the data from the Detailed Artifact databases.
  • Do sophisticated queries or analysis of the artifact data.
  • Do additional analysis on the data in any of the tables.
  • Use the images in another publication (see our Fair Use Policy).
  • Need to format the text differently for presentation (see our Fair Use Policy).

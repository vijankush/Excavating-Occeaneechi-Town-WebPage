Context Catalog

The Context Catalog contains basic information about all of the excavated contexts at the Fredricks site. The catalog is in dBase III format (*.dbf) and can be accessed using most quality database and spreadsheet applications.

Download Context Catalog (72 KB)

Description of the field headings used in the Context Catalog.

Heading Description
CODE The name of the file containing the artifact information for this context (see artifact data files).
PROV The name of the context.
TYPE The type of the context.
ASSOC Associated contexts.
PICT A technical field used by the report software on the original CD-ROM.
ARTPGNO A technical field used by the report software on the original CD-ROM.
LENGTH The length of the context in feet.
WIDTH The width of the context in feet.
DEPTH The depth of the context in feet.
AREA The overall surface area of the context in feet squared.
VOL The volume of the context in feet cubed.
CENTERX The east-west component (i.e., x-ordinate on the site grid) of the center position of the context in feet.
CENTERY The north-south component (i.e., y-ordinate on the site grid) of the center position of the context in feet.
SQCENTER A technical field used by the report software on the original CD-ROM.
HOURS An estimate, based on actual data, of the man-hours it might take to excavate this context. Used by the Electronic Dig chapter, the numbers are based on an abstract calculation involving the type of context and its volume.

Version Information

The authors of Excavating Occaneechi Town (Web edition) recognize that, while all the information recovered in an archaeological excavation has scientific importance, it may be inappropriate to display or view some of that information, such as photographs of human bones, in a public or instructional context. Therefore, we have provided two versions of this edition. The Professional Version provides access to all of the information about the site; the Instructional Version provides access to all of the information except for photographs of human burials.

Native Pottery Data

The Native Pottery Database contains observations and measurements for all analyzed native-made pots and potsherds recovered at the Fredricks site. The database is in dBase III format (*.dbf) and can be accessed using most quality database and spreadsheet applications.

Download Native Pottery Database (1,334 KB)

Description of the field headings used in the Native Pottery Database.

Heading Description
